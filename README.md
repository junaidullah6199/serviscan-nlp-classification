# ServiScan – NLP Classification for Marketplace Structuring

AI-powered text classification system designed to improve semantic organization
of Craigslist service listings.

## Objective
Automatically classify unstructured service titles into five refined categories:
- Body-Work
- Hair-Styling
- Health-Aid
- Fitness
- Others

## Tech Stack
- Python
- Scikit-learn
- XGBoost
- TF-IDF Vectorization
- Pandas
- NLP preprocessing (NLTK)

## Models Implemented
- Logistic Regression
- Random Forest
- XGBoost (Best Performing Model – 82% Accuracy)

## Business Impact
- Improves search efficiency
- Reduces moderation burden
- Enhances user trust
- Scalable backend solution
